# HTML5Application1 Project

## Project Overview

This project is a simple HTML5 web application created using
**NetBeans**. It contains basic HTML files that form the foundation of a
frontend layout.

## Folder Structure

    HTML5Application1/
    ├── nbproject/            # NetBeans project configuration files
    │   ├── project.properties
    │   ├── project.xml
    │   └── private/
    │       ├── private.properties
    │       └── private.xml
    └── public_html/          # Main web content
        ├── index.html
        └── index1.html

## How to Run the Project

1.  Open **NetBeans IDE**.
2.  Go to **File → Open Project**.
3.  Select the folder `HTML5Application1`.
4.  Click **Run** or press **Shift + F6**.
5.  The project will launch in your default browser.

## Requirements

-   NetBeans (HTML5/JavaScript support)
-   Any modern web browser (Chrome, Firefox, Edge, etc.)

## Description of Files

### `index.html`

Main homepage of the project.

### `index1.html`

Secondary page or an alternate version of the homepage.

## Notes
